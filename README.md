# Motor Model

## はじめに

このソースコードはモータに電流を入力して位置が出力されるまでの過程を C++ でモデル化したものです。
このソースコードを作成する過程は下記の Zenn スクラップに記録されています。

### Zenn スクラップ

https://zenn.dev/tatsuyasusukida/scraps/e412338e5f325d

## 実行

```sh
make main && ./main > main.csv
```

## テスト

```sh
make unittest && ./unittest
```
